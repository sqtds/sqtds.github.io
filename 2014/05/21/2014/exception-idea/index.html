exception idea Server is not connected. Deploy is not available.

  • 异常
java

当在idea启动tomcat的时候，报如上错误。经过google，是由于在tomcat的bin目录下设置了java_opts指向了其他方向，导致环境变量改变。

解决的方式就是将自己设置的java_opts去掉即可。
